## 1. 定義

- 「あなた(You)」とは、本プロジェクトに Contribution を提出する個人または法人を指します。
- 「貢献(Contribution)」とは、あなたが本プロジェクトに提出する、あらゆる著作物(ソースコード、ドキュメント、その他の成果物)およびその修正・追加を指します。

## 2. 著作権ライセンスの付与

あなたは、本プロジェクトの管理者(以下「メンテナ」)に対し、あなたの Contribution について、**恒久的・世界的・非独占的・無償・取消不能**の著作権ライセンスを付与します。これには、Contribution およびその二次的著作物を、**現在または将来の任意の OSS ライセンス条件(AGPL-3.0 および将来の Apache-2.0 を含む)で**複製・改変・公衆送信・再ライセンス・頒布する権利を含みます。

この付与により、メンテナは本プロジェクト全体(あなたの Contribution を含む)のライセンスを、OSI 承認のオープンソースライセンスの範囲で将来変更できます。

## 3. 特許ライセンスの付与

あなたは、あなたの Contribution に関して有する特許のうち、Contribution 単体または本プロジェクトとの結合により必然的に侵害されるクレームについて、メンテラおよび本プロジェクトの利用者に対し、**恒久的・世界的・非独占的・無償・取消不能**の特許ライセンス(製造・使用・販売・提供・頒布を許諾)を付与します。

あなたは以下を表明します:

- (a) 各 Contribution はあなた自身が創作したものであり、これを付与する法的権原を有すること。

- (c) 各 Contribution に、あなたが知る限り第三者のライセンス条件が付随する場合は、その旨を明示すること。

Contribution は「現状のまま(AS IS)」提供され、明示黙示を問わずいかなる保証も伴いません。

## 同意方法

Pull Request に以下を記載する(または CLA bot の指示に従う)ことで、本 CLA に同意したものとみなします:

```

Signed-off-by: 氏名 <email@example.com>
```
